## ЦА 1: SEO-специалисты

### Проблема
SEO-специалист ведёт 10-30 клиентов. Клиенты начали спрашивать: "Нас нет в ChatGPT — сделай что-нибудь".
Специалист не знает как это делается, боится потерять клиента, хочет закрыть вопрос быстро и красиво.
Делать GEO вручную = 1-2 часа на сайт. Умножить на 20 клиентов — нереально.

### User Story
> Я SEO-специалист, веду 15 клиентов. Один клиент спросил почему его нет в Perplexity.
> Я хочу за 5 минут дать ему ответ и решение — и лучше если это займёт не мои руки.

### Путь

```
Telegram SEO-чат
→ Вижу пост с GIF: "вставил URL клиента → за 10 сек готов llms.txt + schema"
→ Перехожу по ссылке
→ Лендинг /seo — сразу форма "вставь URL клиента"
→ Вставляю, жду 10 сек → вижу результат (без регистрации)
→ Копирую код → вставляю на сайт клиента
→ Думаю "блин, у меня 14 ещё таких клиентов"
→ Регистрируюсь → dashboard с мультиклиентами
→ На 4й клиент — апгрейд $30/мес
```

### Каналы (автоматизируем максимально)

**Telegram (ручной старт, потом реферальная):**
- Чаты: SEO Talks, SEO без воды, Веб-аналитика и SEO, Продвижение сайтов
- Что постим: GIF (10 сек, URL → результат) + 2 строки текста + ссылка
- Когда: вторник-четверг 11-14 мск
- Как найти чаты: поиск в Telegram "SEO", "продвижение", "интернет-маркетинг" → фильтр 1k+ участников
- Частота: 1 раз в чат, потом только по вопросам

**vc.ru (1 статья, пассивный трафик навсегда):**
- Заголовок: "Как добавить GEO-оптимизацию каждому SEO-клиенту за 2 минуты"
- Структура: проблема → инструмент → результат → ссылка
- Публикуется 1 раз → индексируется Яндексом → вечный трафик

**SEO лендинга (пассивный):**
- URL: `/seo` → ключи: "gео оптимизация инструмент", "llms txt для сайта"
- H1 + мета заточены под запросы

**Indie Hackers:**
- Пост "Show IH: Free GEO optimization tool for SEO agencies"
- Аудитория: разработчики и SEO, которые ищут инструменты

## ЦА 2: Веб-разработчики

### Проблема
Разработчик делает сайт клиенту. Клиент говорит: "Слышал про AI-поиск, сделай".
Разработчик не хочет ручками писать schema.org разметку под каждый проект.
Хочет API который вызвал → получил готовый код → вставил. 5 минут и готово.

### User Story
> Я делаю 5-8 сайтов в месяц на WordPress/Tilda.
> Хочу добавить "AI оптимизацию" как услугу и брать за это +$100 к проекту.
> Мне нужен API который сам всё генерирует — я просто интегрирую.

### Путь

```
GitHub / RapidAPI / dev.to статья
→ Вижу: "GEO Optimization API — 1 request → llms.txt + schema + FAQ"
→ Пробую curl без регистрации (playground)
→ Получаю JSON с готовым кодом
→ Думаю "окей, это работает, беру API ключ"
→ Регистрируюсь → получаю ключ → интегрирую в свой процесс
→ Использую на 3 проектах → на 4й — апгрейд $20/мес
```

### Каналы

**GitHub (пассивный, долгосрочный):**
- Публичный репо с README, примерами, SDK
- Open source клиентская часть → звёзды → Google
- BYOK: разработчик ставит свой OpenAI/Gemini ключ → платит только за платформу

**RapidAPI (пассивный):**
- Листинг бесплатно
- Категория: "SEO Tools", "Content Generation"
- Название: "GEO Optimizer — AI Search Visibility API"

**Hacker News (1 пост Show HN):**
- Формат: "Show HN: API to make any website visible in ChatGPT/Perplexity"
- Время: вторник 9-11 EST
- Важно: живой демо в посте, отвечать на все комменты

**dev.to / Хабр:**
- Статья с кодом: "Auto-generate llms.txt for your clients via API (Python/JS examples)"

**Telegram разработчиков:**
- Чаты: Веб-разработка, WordPress CIS, Tilda Community Dev
- Пост: скрин кода + результат + ссылка на docs

## ЦА 3: Владельцы интернет-магазинов

### Проблема
Владелец магазина слышал что "ИИ убивает Google".
Он не технарь, не знает что такое llms.txt, но хочет чтобы его нашли в ChatGPT.
Агентство просит $500+/мес за GEO — это дорого. Хочет решение которое само работает.

### User Story
> У меня магазин одежды на WooCommerce. Продажи с органики упали.
> Хочу чтобы когда кто-то спрашивает ChatGPT "где купить платье онлайн" — меня предложили.
> Поставить плагин я могу, настраивать сложное — нет.

### Путь (плагин — максимум автоматики)

```
Поиск в WordPress Admin: "AI SEO", "ChatGPT optimization"
→ Находит плагин: "AI Search Optimizer — GEO & llms.txt"
→ Устанавливает 1 кликом
→ Wizard: "введи описание магазина" (или парсится сам)
→ Нажимает "Оптимизировать" → всё применяется автоматически
→ Видит: "✅ llms.txt создан, schema установлена, FAQ добавлен"
→ Freemium → через неделю видит трафик из Perplexity → апгрейд $9/мес
```

### Каналы

**WordPress.org (пассивный, главный):**
- Плагин в директории → органика в Admin поиске
- Теги: seo, ai, chatgpt, schema, llms-txt

**Аутрич в Instagram (ручной, для первых 50):**
- Как искать лидов: ищем магазины в Инстаграм через хэштеги (#интернетмагазин, #shopify, #woocommerce)
- Фильтр: 1k-50k подписчиков, есть сайт в bio, пишут про e-commerce
- Что пишем в DM: "Видел ваш магазин — у вас нет AI-оптимизации, теряете трафик из ChatGPT. Дам бесплатно на 3 месяца если поставите и дадите отзыв"
- Объём: 20-30 DM в день → ожидаем 10-15% отклик → 50 магазинов за 2 недели

**Личный аутрич:**
- Форумы и группы ВКонтакте для владельцев магазинов
- Чаты Telegram: "ecommerce ru", "WooCommerce CIS", "Shopify Россия"
- Oтвечаешь на вопросы про SEO → даёшь ссылку как решение

## ЦА 4: Маркетинговые агентства

### Проблема
Агентство теряет клиентов которые уходят к "ИИ-специалистам".
Хочет добавить GEO как услугу за $300-500/мес, но делать руками дорого.
Нужен инструмент который автоматизирует работу и можно брендировать под себя.

### User Story
> Я владелец SEO-агентства, 20 клиентов.
> Хочу продавать "AI оптимизацию" за $300/мес сверху к SEO.
> Мне нужен инструмент под моим брендом, который я настрою 1 раз и он работает сам.

### Путь

```
LinkedIn / прямое письмо
→ Письмо: "Ваши клиенты спрашивают про ChatGPT — у нас white label решение"
→ Демо-звонок 20 мин → показываем dashboard с их логотипом
→ Trial 14 дней бесплатно
→ Ставят 2-3 клиентам → те видят упоминания в Perplexity
→ Агентство подписывается $150/мес (до 25 клиентов)
```

### Как искать лидов (аутрич)
- SEO агентства: ищем через Яндекс "seo агентство москва/питер"
- vc.ru: ищем авторов которые пишут про SEO → смотрим их агентство
- LinkedIn: "SEO Agency founder Russia/CIS"
- Что отправляем: короткое письмо с конкретным оффером + ссылка на демо

**Письмо-шаблон:**
```
Привет [Имя],

Видел ваше агентство [название].
Ваши клиенты уже спрашивают про AI-поиск?

Мы делаем white label GEO-инструмент: ваш бренд, ваши клиенты, наша технология.
Агентства добавляют $200-500/мес к каждому клиенту.

14 дней бесплатно: [ссылка на демо]
```

## ⭐ Главная ставка: Органический трафик

**Аутрич = только для получения первых 20-50 пользователей и кейсов.**
Основной трафик строится на инструментах которые работают сами:

### Приоритет каналов

| Канал | Тип | Трафик | Усилие |
|-------|-----|--------|--------|
| WordPress.org плагин | Пассивный | ⭐⭐⭐⭐⭐ | Сделать 1 раз |
| SEO лендингов (Яндекс/Google) | Пассивный | ⭐⭐⭐⭐ | Статьи в блог |
| GEO лендингов (ChatGPT/Perplexity) | Пассивный | ⭐⭐⭐ | Разметка сайта |
| RapidAPI маркетплейс | Пассивный | ⭐⭐⭐ | Сделать 1 раз |
| Ecwid / Wix App Market | Пассивный | ⭐⭐⭐ | Сделать 1 раз |
| Product Hunt / Show HN | Разовый буст | ⭐⭐⭐⭐ | Подготовка 1 день |
| Аутрич (TG, Instagram, email) | Ручной | ⭐⭐ | Постоянные усилия |

### Принцип работы

```
Строим продукт → публикуем на маркетплейсах → SEO лендинг
→ трафик идёт сам → аутрич только для первого запуска и кейсов
```

**Каждая публикация на маркетплейсе = постоянный источник трафика.**
Один хороший WordPress плагин с 4.5 звёздами = тысячи установок в месяц без активных действий.
