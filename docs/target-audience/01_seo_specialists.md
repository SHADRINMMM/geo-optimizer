# ЦА 1: SEO-специалисты и SEO-агентства

## Кто это
Фрилансеры и сотрудники агентств, которые занимаются продвижением сайтов клиентов в поисковых системах. Ведут 5–50 проектов одновременно. Хорошо знают технический SEO, умеют работать с инструментами.

## Их проблема
- Клиенты начали спрашивать: "Почему нас нет в ChatGPT / Perplexity?"
- SEO-специалист понимает что GEO — это важно, но нет понятного инструмента
- Делать GEO руками долго: schema.org, llms.txt, FAQ-разметка — это час работы на каждый сайт
- Боятся потерять клиента который уйдёт к тому кто "умеет в ИИ"

## Как они нас найдут (бесплатные каналы)

**Telegram-чаты:**
- Профессиональные SEO-чаты (их десятки с 1k–30k участников)
- Один живой пост с демо: "вставил URL → получил llms.txt + schema за 10 сек"
- Что нужно: демо без регистрации, GIF или скрин результата

**vc.ru / Хабр:**
- Статья "Как добавить GEO-оптимизацию каждому клиенту за 2 минуты"
- Что нужно: 1–2 реальных кейса до/после

**Поиск (SEO на наш лендинг):**
- Запросы: "GEO оптимизация инструмент", "llms.txt генератор", "как попасть в ChatGPT сайт"
- Что нужно: лендинг с правильными ключами + 3–5 статей в блоге

## Что нужно сделать для этого канала
- [ ] Демо-режим без регистрации (вставил URL — увидел результат)
- [ ] Лендинг с фокусом на SEO-специалистов
- [ ] GIF/видео демо для постов в чатах
- [ ] 1 статья на vc.ru или Хабр с кейсом

## Монетизация
- Тариф "Агентский": $30–80/мес
- Мультиклиентский dashboard
- Неограниченное число сайтов
- API доступ

Один клиент которого удержали благодаря GEO = окупает подписку на год вперёд.
