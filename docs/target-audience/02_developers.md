# ЦА 2: Веб-разработчики и технические фрилансеры

## Кто это
Разработчики которые делают сайты под заказ (фрилансеры и небольшие студии). Работают с WordPress, WooCommerce, Tilda, кастомными стеками. Ведут 3–30 клиентских проектов. Ищут готовые решения которые можно быстро внедрить.

## Их проблема
- Клиент спрашивает "почему меня нет в ChatGPT" — разработчик не знает как это закрыть
- Делать schema.org и llms.txt вручную под каждый сайт — трата времени
- Хочется добавить GEO как отдельную услугу и брать за это деньги
- Нужен API или плагин который встраивается в существующий процесс разработки

## Монетизация
- API тариф: $20–50/мес по числу сайтов/запросов
- WordPress плагин: freemium — бесплатно 1 сайт, платно безлимит
- Реселлинг: разработчик платит оптом, продаёт клиентам дороже

### Open Source / BYOK модель (Bring Your Own Key)
- Репозиторий открытый — разработчик может развернуть сам
- Вставляет свой OpenAI API ключ — платит OpenAI напрямую, не нам
- Мы берём небольшую комиссию за платформенную инфраструктуру (hosted страницы, мониторинг)
- Вариант: $5–10/мес фикс за платформенные фичи, генерация за их счёт

**Плюсы этой модели:**
- Низкий барьер входа — технарь не платит за то что может сделать сам
- Доверие — открытый код, нет vendor lock-in
- Виральность — GitHub звёзды, форки, упоминания
- Воронка: попробовал бесплатно → вырос → перешёл на полный тариф

**Как работает технически:**
- Self-hosted версия: разработчик деплоит сам, использует свои ключи
- Cloud версия с BYOK: работает на нашей инфре, но OpenAI ключ свой

## Почему заплатят
Один API-ключ = инструмент для всех клиентов. Экономия 30–60 мин на проект × 10 проектов = окупается за первый месяц. Open source версия привлекает тех кто никогда не заплатил бы за SaaS — но конвертирует через hosted фичи.
